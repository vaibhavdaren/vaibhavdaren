## 🧬 About

Senior Software Engineer with 8+ years building scalable, resilient backend and cloud-native systems. I've worked across the stack at **Red Hat**, **Amazon Web Services**, **Bright Money**, and **BrowserStack** — from DNS infrastructure serving millions of hosts to fintech platforms processing thousands of accounts per second.

I care deeply about **distributed systems**, **caching architectures**, **observability**, and writing code that other engineers actually want to maintain. When I'm not shipping production systems, I'm mentoring the next generation through **Google Summer of Code** and **The Linux Foundation**.
